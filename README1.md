# NovaCart ✨

A modern React + TypeScript product explorer application built with scalable frontend architecture and real-world state management practices.

## 🚀 Features

* Product listing using external API
* Search functionality
* Add to Favorites
* Remove from Favorites
* Redux Toolkit state management
* LocalStorage persistence
* Dark / Light mode toggle
* Toast notifications
* Responsive modern UI
* React Router navigation
* API error handling

---

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Redux Toolkit
* React Redux
* React Router DOM
* Axios

## ⚡ Installation & Setup

Clone the repository:

```bash
git clone <repository-link>
```

Navigate into the project:

```bash
cd novacart
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🎯 Key Technical Decisions

* Redux Toolkit was used for scalable global state management.
* LocalStorage was implemented to persist favorites after page refresh.
* React Router was used for clean page navigation.
* Axios was used for API handling.
* Component-based architecture was followed for maintainability and scalability.
* Modern UI design with dark/light theme support was implemented for better user experience.
